NERT tool

The NCA-Extended Regression Table (NERT) is a tool for combining NCA results with regression-based results in a causal-pluralism multimethod study. It extends the conventional regression table by adding two columns: the necessity effect size and its p-value for those predictors hypothesized and shown to operate as necessary conditions. NERT allows the results of NCA and regression-based methods (e.g., multiple linear regression, SEM) to be compared and interpreted jointly. See Section 11.3.4 in the book.

NEST tool

The NCA-Extended Solution Table (NEST) is a tool for combining NCA results with QCA results in a causal-pluralism multimethod study. It extends QCA's solution table by adding the required membership score of necessary conditions identified by NCA to each sufficient configuration. This allows the combined interpretation of NCA's necessity results and QCA's sufficiency results. See Section 11.6.3 in the book.

BIPMA tool

The Bottleneck Importance Performance Map Analysis (BIPMA) is a tool for providing evidence-based practical advice on prioritizing actions and resources, based on the combined results of NCA and SEM. It displays predictor variables as points in a map combining their necessity (NCA) and average effect (SEM) results, helping practitioners identify which conditions to address first to achieve a target outcome. See Section 11.4.1.7 in the book.

SCoRe checklist

The SCoRe checklist can be used to evaluate the quality of conducting and reporting an NCA study. A minimum SCoRe-score of 60 is required for a publishable study. In addition, the checklist provides targeted recommendations for improvement. See Section 10.3 in the book.

Interactive SCoRe checklist

AI-SCoRe

AI-SCoRe is a series of prompts that can be used to get AI-generated feedback on the quality of conducting and reporting an NCA study. Application of the tool requires that a publication about the study (e.g., a draft article for a journal) is uploaded to a GenAI model like Claude or ChatGPT. Permission of author(s)/publisher is needed to upload a publication. Without such permission, the Interactive SCoRe checklist for human evaluation can be used. Always check the AI-generated results.

Quick Start Guide NCA with Stata

This guide helps to get started with condunting a Necessary Condition Analysis (NCA) data analysis in Stata using the nca package. It covers installing the nca package, running the core analysis functions, and interpreting the results. Please note that access to the Stata software is required to use the nca package. See Appendix B in the book.

Quick Start Guide NCA with R

This guide helps to get started with the NCA package in R (version 5.0.0). It covers installing and loading the package, running the core functions such as nca_analysis and nca_output, and interpreting the results. No prior experience with R is required to follow the guide. See Appendix B in the book.

Demonstration NCA with R

This R script demonstrates how to perform the data analysis of Necessary Condition Analysis (NCA) using the NCA package in R. It walks through a complete example — loading data, running the analysis, and interpreting the output — and can be used as a hands-on starting point for your own NCA applications. See Chapter 9 in the book.
