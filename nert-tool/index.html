NCA-Extended Regression Table NCA + regression · multimethod

Instruction

Use results of the NCA and regression (or SEM) analyses as input to the tool. The tool produces the NERT table. An internet connection is required for downloading as PNG or PDF. The tool may not work on all platforms (contact us if you encounter a problem). The backgrounds of the NERT tool are described in Section 11.3, Section 11.4, and Appendix H in Dul, J. (2026), Necessary Condition Analysis (NCA): Principles and Application. Chapman & Hall / CRC Press.

▼ How to produce a publication-quality table

For a publication-quality table, click Download .R and run the script in an R Markdown (.Rmd) document in RStudio. This produces a LaTeX table with vector text and correct formatting. Required packages: knitr and kableExtra. A LaTeX installation is needed (e.g. TinyTeX: tinytex::install_tinytex()). Paste the script as a code chunk with options echo = FALSE, results = 'asis'. The PNG and PDF download buttons produce raster previews only, suitable for presentations but not for journal submission.
